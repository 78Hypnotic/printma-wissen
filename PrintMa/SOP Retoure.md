# SOP: Retoure

> Ablauf bei DHL Retouren

## Retoure einleiten

### Kunde fordert Retoure an
1. Kunde meldet sich via Email/TikTok/Amazon
2. Grund erfassen:
   - Defekt/Beschädigt
   - Falsches Produkt
   - Anders als erwartet
   - Besserer Preis gefunden
   - Sonstiges

### Retouretikett erstellen (DHL GKP)
1. DHL GKP Portal → Retoure → Neues Label
2. Empfänger: **PrintMa, Randenstr. 8, 78234 Engen**
3. Absender: Kundenadresse
4. Label als PDF downloaden
5. Label per Email an Kunden senden

## Ablauf

```
Retoure erhalten
├── Paket öffnen + prüfen
├── Zustand dokumentieren (Fotos)
├── Grund erfassen
│
├── Zustand OK
│   ├── Gutschrift an Kunden
│   └── Ware ggf. wieder einlagern
│
└── Zustand nicht OK
    ├── Reklamation bearbeiten
    ├── Ersatzlieferung (wenn Material da)
    └── Kundenservice kontaktieren
```

## Entscheidungsmatrix

| Grund | Aktion |
|---|---|
| Defekt (unser Fehler) | Kostenlose Ersatzlieferung + Retoure kostenlos |
| Falsches Produkt (unser Fehler) | Neues Produkt + Gratisretoure |
| Anders als erwartet | Retoure auf Kundenkosten, ggf. 50% Wertersatz |
| Beschädigt (Versand) | DHL Claim, kostenlose Ersatzlieferung |

## Retoure-Tracking

- [ ] Retouretikett gesendet
- [ ] Paket beim Kunden
- [ ] Paket bei PrintMa eingegangen
- [ ] Retoure geprüft
- [ ] Gutschrift/Ersatzlieferung eingeleitet
